# Hande Pehlivan — Portfolio

This repository hosts the GitHub Pages portfolio for Hande Pehlivan, showcasing data and research analytics projects with a focus on healthcare, customer insights, and predictive modeling.

## Portfolio summary
A curated list of selected projects featured on the portfolio site.

### 1. Predicting Diabetes Risk from Behavioral & Medical Factors
- CDC BRFSS 2021 analysis of 134,702 respondents.
- Identified demographic, behavioral, medical, and economic predictors of type II diabetes risk.
- Includes data cleaning, LASSO feature selection, model comparison, and public health interpretation.
- GitHub: https://github.com/hpehlivantopcu/Diabetes-probability-based-on-behavioral-medical-demographic-and-economic-factors

### 2. Customer Churn Analysis & Prediction
- End-to-end churn analysis on 2M+ telecom customer records.
- Built feature engineering, segmentation, and predictive classification models.
- Delivered retention strategy recommendations and monthly trend dashboards.
- GitHub: https://github.com/hpehlivantopcu/churn-analysis

### 3. Airbnb Pricing Engine + A/B Test Simulation
- Developed an Airbnb pricing model for NYC listings.
- Included simulation of A/B tests to estimate revenue lift and validate treatment performance.
- Highlighted model-driven pricing recommendations.
- GitHub: https://github.com/hpehlivantopcu/airbnb-pricing

### 4. Heart Disease Prediction (ML Comparison)
- Compared multiple classifiers on the UCI Heart Disease dataset.
- Evaluated KNN, Logistic Regression, SVM, Naive Bayes, Random Forest, and Neural Network.
- Produced ROC analysis, confusion matrices, and accuracy benchmarking.
- GitHub: https://github.com/hpehlivantopcu

### 5. Pandemic Flu Spread Simulation
- Simulated influenza spread in a classroom using a discrete SIR model.
- Ran 100K Monte Carlo iterations and validated results against theoretical distributions.
- Used simulation to compare outbreak scenarios.
- GitHub: https://github.com/hpehlivantopcu/flu-simulation

### 6. Statistical Analysis Dashboard
- Built an analytics dashboard for business KPIs including conversion, churn, and revenue metrics.
- Designed for executive storytelling with trend visualizations and segment insights.
- GitHub: https://github.com/hpehlivantopcu
